# Example prompts

Try these in a Claude Code session after installing the skill. Each one
should trigger which-llm and produce a substantive, data-grounded answer.

## Recommendations

> *"I'm building a chatbot with image input on a budget. Which model has the
> best intelligence per dollar with image support?"*

> *"Give me three good open-weights options for code generation."*

> *"What's the cheapest reasoning model with at least 256k context?"*

## Comparisons

> *"Compare Claude Opus 4.7, GPT-5.5, and Gemini 3.1 Pro on cost and
> benchmarks."*

> *"How does DeepSeek V4 Pro compare to Grok 4.3 at the high effort tier?"*

## OpenRouter lookups

> *"What's the OpenRouter slug for Claude Sonnet 4.6?"*

> *"Is there a free version of DeepSeek V4 Flash? If so what's the slug?"*

> *"List all free models on OpenRouter."*

## Frontier exploration

> *"Show me the cost-vs-intelligence Pareto frontier for models under $250."*

> *"Plot the Pareto frontier for vision-capable models."*

## Data freshness

> *"Refresh the which-llm dataset and tell me what changed."* (does a full
> re-scrape)

> *"How fresh is the which-llm data?"*
